Web Developer

I am a front-end developer living near Philadelphia, Pennsylvania (able and available to relocate anywhere) and I'm looking to help your organization build and maintain modern responsive web pages. I have focused on building applications using React on the front-end, SQL (or postgresql/sqlite) on the backend and node.js to handle traffic. I've also built scrapers and tools using Python3, Ruby (RoR), Angular, and ASP.NET. Professionally I've worked at a business-to-business CMS company (KSA&D) where I contributed mostly to satisfy the front-end needs of clients but I also co-developed the City of Bethlehem's new site and made PHP forms for client listservs. All of my work is on my github and I'm comfortable using git from the command line.

I have a B.A. in Political Science and Philosophy and 40 credits earned towards a PhD in School Psychology. I've found that my educational background has given me excellent command of the english language and critical thinking skills that have allowed me to contribute to projects and decisions sometimes beyond the scope of my position.

My web development career began in 2016 with the help of a long-time friend and senior full-stack developer. In 2017 I attended a highly selective (less than 3% acceptance rate) 6-month 80hrs/week full-stack web development training program in San Francisco called appAcademy where I built full-stack apps in a collaborative environment using Rails, SQL, React, and Node.js. At appAcademy I went "under the hood" to understand and implement built-in javascript and ruby methods, I learned about web sockets and built servers running node.js, I used metaprogramming to gain insight into complex programming practices, and implemented higher-level computer science topics such as stacks and queues, recursion, and sorting.

Check out some of my work below and please don't hesitate to contact me; I'm looking forward to connecting with you!

Projects

  • Youtube Clone

    I developed this youtube clone with React and the Redux library. I used the "create-react-app" library for easy transition to hosting on Github Pages. This application is very very fast. I designed the search bar to make youtube API calls as you type so you'll see the application populate relevant videos even before you're finished typing. Making API calls in this manner is not efficient at all and wastes a lot of resources but it's so fast and this is a small personal application so I kept it as-is because, if anything, it's neat to see how quickly the browser can send and receive data.

  • Weather Essentials

    A weather application that I made using React. It uses the DarkSky weather API to perform an asynchronous call to locate your current location then the application renders a succinct overview of current weather conditions in addition to two nice visualizations of how the weather in your area will be in 12, 24, and 48 hours

  • Dictation Application

    I put this together using javascript and basic HTML/CSS. This application will record speech in real time assuming you have a microphone available. Initially I had run into a problem with the blazing-fast speed in which the application would try and render a user's speech on the screen. I found that by including a few timeouts I could delay the program's lookup time and improve the accuracy of the speech recorded. I even trained it to incorporate basic punctuation so the rendered text looks more representitive of proper sentence structure.
